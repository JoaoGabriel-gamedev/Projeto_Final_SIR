<p>
  Meu protejto é a Simulação de Epidemias (SIR), um modelo que calcula matematicamente a propagação de uma doença. O projeto conta com bibliotecas como a Numpy, Matplotlib e a Scipy e é costruido inteiramente em python. Utiliza variáveis como população total, infectados, recuperados/removidos, taxa de trasmissão da doença, taxa de recuperação dos infectados e o período de tempo da propagação. Com esses dados, o programa cria um gráfico que mostra os valores de infectados, suscetíveis e recuperados no período proposto. <br>
  O projeto está no arquivo Simulador_epidemia.ipynb. Para modificar os dados, é preciso que o arquivo seja baixado no dispositivo e utilizando uma plataforma, como o Jupterlab, alterar os dados diretamente no código.
